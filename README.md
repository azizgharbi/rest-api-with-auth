### API REST ( Minimalism starter without database client)

- Nodejs
- Express
- Typescript

#### Routes

- `/login`
- `/register`
- `/secret`: `protected` 

- Protected routes: `{ headers:{ authorization: <Token> }` the token will expire in 10 minutes.
